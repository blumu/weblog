SumatraPDF for TeX users


I have contributed to the Sumatra PDF viewer project by developing new features targeted at TeX users. Most of them (autorefresh, DDE commands, pdfsync backward and forward synchronization, synctex synchronization) are now integrated into the official release, but I continue to release my custom build on this page. Currently, the only difference with the official release is the presence of a dialog box to let the user configure the inverse-search feature.
